# Twitter/X Clone

A modern, responsive **Twitter/X-inspired social media interface** built to demonstrate front-end engineering, responsive web design, reusable components, and clean UI architecture. This project recreates the look and feel of a contemporary social platform while emphasizing accessibility, maintainable code, and an engaging user experience.

---

##  Overview

This project was created to strengthen practical front-end development skills by recreating the interface and interactions of a modern social media platform.

It showcases responsive layouts, intuitive navigation, modern UI design, and scalable project organization while following web development best practices.

---

##  Features

-  Responsive design for desktop, tablet, and mobile
-  Twitter/X-inspired home feed
-  User profile interface
-  Interactive engagement buttons
-  Search and navigation layout
-  Trending section
-  Reusable UI components
-  Smooth hover animations and transitions
-  Semantic, accessible HTML structure
-  Clean and maintainable codebase
